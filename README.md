# Pizza Sales Analysis Project

## Project Overview
This project performs a deep dive into a year's worth of pizza sales data to uncover seasonal trends, popular categories, and menu optimization opportunities.

## Key Findings
- **Peak Sales:** July recorded the highest revenue, followed by a seasonal dip in October.
- **Top Category:** The Thai Chicken pizzas remain the primary driver of volume.
- **Optimization:** Identified 5 underperforming pizzas that could be replaced to reduce inventory waste.

## Technologies Used
- **Language:** Python 3.x
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
